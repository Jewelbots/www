#Jewliebots WWW Site
This is a static site that contains the Jewliebots homepage. Its initial release date is late April.

To 'build' the site:

On MAC OS X:
 
Install Homebrew

`/var/src/www` is the default location for these static files to be served. 

Run this command:

`chmod +X ./bootstrap.sh`  
`./bootstrap.sh`
`./build.sh -e dev`
